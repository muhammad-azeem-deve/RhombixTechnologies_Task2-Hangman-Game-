# 🎯 Hangman Game (Flask Web Application)

A modern and interactive **Hangman Game** built using **Python and Flask**. This web-based application allows users to guess letters, track remaining attempts, and enjoy a smooth gameplay experience with an attractive user interface.

---

## 🚀 Overview

This project is a simple implementation of the classic Hangman game, designed to demonstrate **Flask fundamentals**, session handling, and dynamic UI rendering. It is ideal for beginners learning web development with Python.

---

## ✨ Features

- 🔤 Predefined word list for gameplay  
- 🎮 Interactive letter-by-letter guessing  
- 📉 Attempts tracking system  
- 🎉 Win and Lose conditions  
- 🔄 Restart game functionality  
- 🎨 Clean and responsive dark-themed UI  

---

## 🛠️ Tech Stack

- **Backend:** Python (Flask)  
- **Frontend:** HTML, CSS  
- **Session Management:** Flask Sessions

## ⚙️ Installation & Setup

### 1. Clone the Repository

- git clone https://github.com/muhammad-azeem-deve/RhombixTechnologies_Task2-Hangman-Game-
- cd hangman-flask
### 2. Install Dependencies
- pip install flask
### 3. Run the Application
- python app.py
### 4. Open in Browser
- http://127.0.0.1:5000/
### 🎮 How to Play
- Enter a single letter in the input box
- Correct guesses reveal the hidden word
- Incorrect guesses reduce remaining attempts
- Win by guessing the word before attempts reach zero
- Use the restart button to play again
